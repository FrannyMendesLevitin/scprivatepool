MovieMarkers

A basic and experimental reader for movie markers such as those by FCP and Cinelarra


superclass: Object


*new(path, type)


path - a string with the path to a .xml file

type - a symbol with the kind of xml to read

the implemented types are:

'fcp' - Final Cut and iMovie xml export (also available in Adobe Premiere)Â 

'cinelarra' - Cinelarra xml file

NOTE: It's not possible to create markers from within iMovie, so the markers are always an empty array.

// example

// clips (iMovie & FCP)

a = MovieMarkers("iMovie.xml".resolveRelative, \fcp);

a.clipFrames; // clip start positions (frames)

a.clipTimes; // clip start positions (secs)

a.clipNames; // clip names (undefined = filename)

a.clipDurations; // clip durations (secs)

a.clipLengths; // clip lenghts (frames)

a.markerFrames; // usually a [ ] for iMovie xml files


// markers (FCP only)

a = MovieMarkers("fcp.xml".resolveRelative, \fcp);

a.markerNames;

a.markerFrames; // marker start position (frames)

a.markerTimes; // marker start position (secs)

a.markerDurations; // usually nil if marker is not a region

a.markerLengths; // usually -1 if marker is not a region


[a.markerNames, a.markerTimes].flop; // useful pairing


// cinelarra

// labels (markers)

a = MovieMarkers("cinelarraMarkers.xml".resolveRelative, \cinelarra);

a.markerFrames;

a.markerNames;

a.markerTimes;

// edits

a = MovieMarkers("cinelarraCuts.xml".resolveRelative, \cinelarra);

a.clipNames; // not yet implemented

a.clipTimes; // all tracks, all cuts

a.clipTimes[0]; // clip start times track 1

a.clipFrames[1]; // clip start frames track 2

a.clipDurations; // clip durations (secs) all tracks

a.clipLengths[1]; // clip lengths (frames) of track 2


a.clipTimes.flatten.asSet.asArray.sort // get all the cuts from all tracks
